## 📖 About

Backend REST API powering **SportPal** — a mobile app that helps you find sport partners nearby. Built with Node.js/Express and MongoDB, it handles auth, user matching, media uploads and email notifications.

---

## ✨ Features

- 🔐 JWT authentication + bcrypt password hashing
- 👥 Sport partner matching system
- 📸 Image/media uploads via Multer + Cloudinary
- 📧 Email notifications via Nodemailer
- 📍 Geolocation-based partner search
- 🔒 Secure password reset flow

---

## 🛠️ Stack

| Technology | Usage |
|---|---|
| Node.js + Express | Server framework |
| MongoDB + Mongoose | Database & ORM |
| JWT + BCrypt | Auth & security |
| Cloudinary + Multer | Media uploads |
| Nodemailer | Email notifications |
| Dotenv | Environment config |

## 🚀 Run locally

```bash
git clone https://github.com/mohamedmmc/SportPal-Backend.git
cd SportPal-Backend
npm install
cp .env.example .env   # add your env vars
npm start
```
